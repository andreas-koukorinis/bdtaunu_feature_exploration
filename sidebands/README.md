Sideband plots
===

Sideband region:
---
+ eextra >= 3 GeV.
+ mmiss2prime <= -15 GeV.
+ sig\_hp3 >= 2.3 GeV.

Data for plots:
---
+ `brf_others.dat` and `offpeak_weights.csv` already exist in the directories.
+ Data files such as `mc_sideband.csv` are not uploaded and must be downloaded using `download_events.sql`.
